# Benign Templates Directory

Place your downloaded templates here.

## Structure

```
public/benign-templates/
├── restaurant/
│   ├── index.html
│   ├── css/
│   ├── js/
│   └── images/
├── corporate/
│   └── ...
├── tech/
│   └── ...
├── realestate/
│   └── ...
└── healthcare/
    └── ...
```

## How to Add Templates

1. Download templates from [themewagon.com](https://themewagon.com)
2. Extract each template
3. Place in corresponding folder (e.g., `restaurant/`, `corporate/`)
4. Ensure each folder has an `index.html` file

## Template Requirements

- Must have `index.html` as entry point
- Can include CSS, JS, images subdirectories
- Should be fully functional standalone websites
- No external dependencies (or bundle everything)
